## ■Java開発環境構築（SVN）
    ①SubVersionの最新のインストーラと日本語パックをＤＬしてインストール。
    　SVNのアカウントで取得対象のリポジトリが表示できること
    ②JDKインストールor配置(例：C:\Program Files\java\配下)
    ③システム環境変数「JAVA_HOME」にJDKのインストールフォルダを設定(binフォルダではない)
    　※必要に応じて、システム環境変数「path」にJDKのbinフォルダを設定
    ④TomEE(apache_tomcat)をprogram_files配下等に配置
    ⑤Pleiades配置orインストール(バージョンにより異なる)
    　※Cドライブ直下でいい
    ⑥eclipse設定
      1. パースペクティブからSVNリポジトリー・エクスプローラーを開き、「新規」－「リポジトリー・ロケーション」を選択。
      リポジトリのURL、ユーザ、パスワードを入力して設定する。
      ※「認証の保管」チェックボックスもチェックする。
      trunkまたはbranchのsrcフォルダを選択して「検索/チェックアウト」を選択する。
      「名前を指定してプロジェクトとしてチェックアウト」を選択し、名称を設定して「完了」ボタンを押下する。
      2. パッケージ・エクスプローラーでは下矢印ボタンを押下し、「フィルター」で「*リソース」の除外を外す
      (.projectファイルが見えるようにする)
      3. eclipseの「ウィンドウ」－「設定」メニューを開き、「Java」－「インストール済のJRE」で「追加」ボタンを押す。
      ※標準VMを選択し、JREホームに、②でインストールしたJDKのトップフォルダを設定
      4. Tomcatサーバの設定
      ※この手順はeclipseが2020年以降のバージョンだと不要
      メニューバーから「ウィンドウ」-「設定」を選択。
      「Tomcat」を選択し、バージョンとTomcatのインストール先ディレクトリ(トップディレクトリ)を選択。
      5. サーバの設定
      メニューバーから「ウィンドウ」-「設定」を選択。
      「サーバー」-「ランタイム環境」を選択し、「追加」ボタンを押下。
      付属のTomcatの各バージョンが表示されるが、④でインストールしたTomcatを使う場合には「追加」ボタンを押下。
      Tomcatのバージョンを指定して「次へ」ボタンを押下
      サーバーの名前とTomcatのインストールディレクトリ(トップディレクトリ)、JREには②でインストールしたJDKを指定して「完了」ボタンを押下。
      6. ビルド環境の設定
      パッケージ・エクスプローラーでプロジェクトを右クリックし、「プロパティー」を選択する。
      「Javaのビルド・パス」の「ソース」タブで必要に応じて「(trunk)/src」を除去し、「(trunk)/src/java」をフォルダー追加する。
      また、デフォルト出力フォルダーは「(trunk)/web/WEB-INF/classes」に変更する。
      「ライブラリー」タブでは「JARの追加」ボタンを押し、チェックアウトしたソースから「web/WEB-INF/lib」配下のライブラリを追加する。
      また、「ライブラリーの追加」ボタンを押し、「サーバー・ランタイム」で上記5で登録したサーバーを追加する。
      必要に応じて「外部JARの追加」ボタンで外部JARを追加する。※事前にローカルにJARをコピーしておくこと。
      「ライブラリー」タブの「JREシステム・ライブラリー」は「編集」ボタンを押し、「代替JRE」に②でインストールしたJDKを指定する。
      7. サーバーの設定
      メニューバーから「ウィンドウ」-「ビューの表示」-「その他」を選択し、「サーバー」を開く。
      右クリックで「新規」メニューを選択し、「サーバーのタイプを選択」に実行するTomcatのバージョンを指定し、
      「サーバー・ランタイム環境」には、5で追加したサーバを選択し、「次へ」ボタンを押下する。
      「追加および除去」画面が開くので、「使用可能」欄のサーバを「構成済み」へ「追加」ボタンで移動させて「完了」ボタンを押す。
    
    DB接続先のconfファイルやメッセージプロパティーなどはPJに応じて修正する。
    
    
### ■マージ  ※ブランチをtrunkにマージ
    ①ブランチのヒストリーでマージしたい開始リビジョンと終了リビジョンを確認。
    [チーム]＞[リソース・ヒストリーに表示]で確認可能。
    ②パッケージ・エクスプローラーでtrunkを右クリックし、[チーム]＞[マージ]で画面を表示。
    ③マージの設定項目を入力する。
    From：URLにブランチのプロジェクトを入力
    リビジョン：開始リビジョン番号を入力
    To：「FromのURLを利用」にチェックのままでOK
    リビジョン：終了リビジョン番号を入力
    ※最新リビジョンまでのマージの場合は、「ヘッド・リビジョン」選んでもOK
    
    
### ■タグ付け
    ①SVNリポジトリ－・エクスプローラー・パースペクティブを開く。
    ②trunkのsrcを右クリックし、[新規]＞[タグ]を選択する。※docなども含める場合はtrunkで右クリックする。
    ③タグ名称とコメントを入力し、OKボタンを押下する。リビジョンはデフォルトのHEADリビジョンで問題ない。
